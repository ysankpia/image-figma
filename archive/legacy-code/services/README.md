# Services Directory

Current product work belongs in:

```text
apps/slice-studio/
```

This directory contains superseded product surfaces, reference services, fallback tooling, and historical experiments. It is not the default Slice Studio runtime.

Before modifying or reviving code here, read:

```text
docs/engineering/current-code-map.md
docs/engineering/legacy-code-inventory.md
```

Default rule:

```text
new product work -> apps/slice-studio
services/* -> reference/deferred/fallback unless an active plan explicitly targets it
```
