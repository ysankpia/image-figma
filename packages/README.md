# Packages Directory

Current product work belongs in:

```text
apps/slice-studio/
```

Packages here are deferred runtime assets, shared contracts, or historical renderer/schema code. They are kept because older Draft/plugin work and validation references may still need them.

Before changing package behavior, read:

```text
docs/engineering/current-code-map.md
docs/engineering/legacy-code-inventory.md
```

Do not route new Slice Studio product behavior through these packages unless an active plan explicitly makes that package part of the current delivery path.
