# ADR 0002: 复杂区域优先 fallback 而不是追求全可编辑

- 状态：accepted
- 日期：2026-05-16

## Context

输入是 PNG。系统无法可靠知道复杂 Banner、插图、图表、多色图标、mask 或光效的真实设计结构。强行拆成大量节点会导致生成慢、错位、图层混乱，并增加失败概率。

## Decision

v0.1 优先保证视觉完整和稳定生成。主要文字、基础 shape、主要图片尽量可编辑；复杂区域允许裁切成图片 fallback。

## Consequences

好处：

- 主链路更稳定。
- 节点数量可控。
- 复杂页面不会因为局部区域拖垮整页。

代价：

- v0.1 不承诺 100% 可编辑。
- 复杂区域后续若要结构化，需要单独迭代识别和渲染能力。
