# 203 单人版工作台偏好设置

## Summary

当前 `/settings` 只是账号会话页，不能配置 Review Workbench 的日常使用偏好。单人版先不做支付、管理端、团队、订阅和复杂账号中心；最有价值的是把工作台常用显示和默认行为收进本地偏好，让用户能按自己的工作习惯开关底部状态栏信息和默认切图模式。

## Scope

本轮新增最小个人偏好：

- 默认裁切模式：`rect`、`subject`、`card`。
- 进入工作台时右侧检查器默认是否折叠。
- 进入工作台时资产列表默认是否折叠。
- 底部状态栏显示项：页面尺寸、缩放比例、保存/任务状态。
- 保留账号信息和退出登录。

偏好先存在浏览器 `localStorage`，不新增数据库表、不改账号 API、不做跨设备同步。当前产品还是单人可用性优先，后续再考虑把偏好同步到用户账号。

## Acceptance

- `/settings` 能修改上述工作台偏好。
- Review Workbench 打开时读取这些偏好。
- 底部状态栏根据偏好隐藏/展示尺寸、缩放和状态。
- 默认裁切模式影响新框和页面级裁切模式默认值。
- 偏好数据损坏时回退默认值，不导致页面崩溃。

## Validation

```bash
pnpm run check
pnpm run build
git diff --check
```

浏览器检查：

- 登录后进入 `/settings` 能看到账号和工作台偏好。
- 修改底部状态栏显示项后，进入 review 页面可见状态栏对应项变化。
- 修改默认裁切模式后，review 详情面板默认模式随之变化。
